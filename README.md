# mathml-tools (MathML 수식 작성 도구 리포지토리)

여기에 있는 파일들을 확인해 보세요! 파일 목록은 다음과 같습니다:

* `index.html`: 이 파일을 HTML로 변환한 것입니다.
* `mathml-editor.html`: MathML 수식 편집기입니다.
* `mathml-grapher.html`: GeeksForGeeks에서 가져온 그래프 계산기에서 MathML을 곁들인 그래프 계산기입니다.
* `mathml-parser.js`: MathML 파서(MathML Parser, MathML을 파싱해서 JavaScript로 나타내는 스크립트)입니다.
